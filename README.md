# Sales Program
Description
This Python program lets the user enter products with their price and quantity. The program stores the information and calculates the total cost.

# How it works
1.The user enters a product name, quantity, and price.
2.The program saves the product in a dictionary.
3.If the product already exists, the quantity is added.
4.The user can keep adding products.
5.At the end, the program shows a summary and the total price.
